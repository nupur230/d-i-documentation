                                                                                                                 #Minio server installation



1. sudo wget https://dl.min.io/server/minio/release/linux-amd64/minio                         -->  Downloads the Minio binary using the wget command.

2. sudo chmod +x minio                                                                                               -->  Makes the downloaded binary executable.
                                                                                             
3. sudo mkdir /var/lib/minio                                                                                           -->  Creates a directory to store Minio's data.

4. sudo mv minio /usr/local/bin                                                                                      -->  Moves the downloaded binary to the system's /usr/local/bin directory.

5. sudo adduser --system --home /var/lib/minio --no-create-home --shell /bin/false minio   --> Creates a system user named minio and a group named nogroup.
    - output
	Adding system user `minio' (UID 114) ...
    Adding new user `minio' (UID 114) with group `nogroup' ...
    Not creating home directory `/var/lib/minio'.
	
6. sudo groupadd minio                                                                                                  -->  Creates a group named minio.

7. sudo mkdir -p /var/lib/minio/data                                                                                -->  Creates a directory for Minio's data and sets the ownership to the minio user and group.

8. sudo chown minio:minio /var/lib/minio/data                                                                -->  Changes user & usergroup of /var/lib/minio/data to as minio.

9. sudo minio server /var/lib/minio/data  &                                                                      --> Starts the Minio server, using the /var/lib/minio/data directory as the storage location.

NOTE : 

1. '&' at the end of command is used to execute the command in background/detached mode.

2. Copy IP address of your server and utilize the port no. 9000

    e.g : http://your-ip:9000
	
3. Edit the inbound rules in security group of your server, enabling the port 9000 

4. Earlier we needed to mentioned the random port generated by minio system to access the mini server webpage, now just mention your IP and port 9000, it will get redirected to the system generated port no.
